# Oracle Enterprise Manager

> Oracle Enterprise Manager (OEM) is an on-premises management platform for Oracle IT infrastructure and applications. Its REST APIs and the EM CLI provide programmatic access to monitoring, administration, and automation — targets, metrics, incidents, blackouts, credentials, users/roles, jobs, patching, deployment procedures, and database lifecycle management. APIs are served from the customer's own Oracle Management Service (OMS) at https://{emHost}:{emPort}/em/api using HTTP Basic authentication.

## APIs

- [Cloud Control REST API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/APIOverview.html): Monitoring, configuration, and administration — targets, metrics, incidents, blackouts, credentials, users, deployment procedures.
- [EM CLI (emcli)](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emcli/): Scriptable command-line interface over Enterprise Manager functionality.
- [Job System API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Create, manage, and monitor jobs and tasks.
- [Metric and Monitoring API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Performance metrics, latest values, and metric group metadata.
- [Target Management API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Discovery, configuration, and lifecycle of monitored targets.
- [Incidents and Events API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Search, view, and manage incidents and events.
- [Blackout Management API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Create and manage maintenance windows.
- [Credentials Management API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Named, monitoring, and preferred credentials.
- [User Management API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Users, roles, privileges, and secure resources.
- [Database Patching and Maintenance API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emlcm/patch-history.html): Fleet patching, Gold Images, compliance.
- [Deployment Procedure API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emrest/rest-endpoints.html): Create, submit, and track deployment procedures.
- [SQL Execution REST API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emmon/executing-sql-rest-api.html): Execute SQL against database targets and the EM repository.
- [Data Guard Administration REST API](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.5/emlcm/data-guard-administration-rest-api1.html): High-availability switchover/failover operations.
- [Cloud APIs (DBaaS)](https://docs.oracle.com/en/enterprise-manager/cloud-control/enterprise-manager-cloud-control/13.4/emclo/using-cloud-apis.html): Self-service database provisioning and quota administration.
